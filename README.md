## 🎯 Strategy

**Entry Conditions:**
- Confluence score ≥ 72 (multiple technical factors aligned)
- Trend, momentum, volume, structure all bullish
- Maximum 3 concurrent positions

**Exit Conditions:**
- Confluence score drops below 50
- Momentum weakening detected
